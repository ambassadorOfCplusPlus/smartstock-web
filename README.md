# smartstock-web — сайт продажи SmartStock

Статический многостраничный сайт (чистые HTML/CSS/JS, без сборщиков и
зависимостей) для продажи SmartStock Desktop и SmartStock Server.

## Страницы

| Файл | Что это |
|---|---|
| `index.html` | Главная: для кого, возможности, две версии, прайс-тизер |
| `desktop.html` | Десктоп-версия: возможности по разделам, скриншоты, требования, честные рамки |
| `server.html` | Серверная версия: как работает синхронизация, роли, сравнение с десктопом |
| `price.html` | Прайс-лист: тарифы десктопа, серверное внедрение, что входит всегда |
| `buy.html` | Как купить: 4 шага + реквизиты СБП + защита от мошенников |
| `license.html` | Лицензионное соглашение (простая бессрочная лицензия) |
| `faq.html` | Частые вопросы (SmartScreen, доверие, данные, 54-ФЗ, маркировка…) |
| `contacts.html` | Контакты + mailto-форма без бэкенда |
| `assets/style.css` | Все стили (палитра в `:root`, адаптив до мобильных) |
| `assets/site.js` | Бургер-меню, активный пункт навигации, год в подвале |

## Бета-версия (кнопка «Попробовать бесплатно»)

`files/SmartStock-2.3.0-Beta-Setup.exe` — установщик бета-версии (лимиты:
10 товаров, 2 дня с первого запуска, затем предложение купить). Кнопки
скачивания: hero на главной, desktop.html, карточка «Знакомство» в прайсе.
Пересборка беты в репозитории `smartstock`:
```
cmake -B build-trial ... -DSMARTSTOCK_TRIAL=ON -DBUILD_TESTS=OFF
cmake --build build-trial --config Release --target smartstock
pwsh -File packaging\build_installer.ps1 -BuildDir build-trial -Suffix "-Beta"
```

## Осталось перед публикацией

1. Перечитать `license.html` (редакция/дата) и при желании цены в `price.html`.
2. Залить папку на хостинг (см. ниже) — и можно давать ссылку в объявлениях.

## Как посмотреть локально

Просто открыть `index.html` в браузере — сайт полностью статический.

## Как опубликовать

Любой статический хостинг:
- **GitHub Pages**: репозиторий → Settings → Pages → ветка `main`, папка `/`.
- Обычный хостинг/VPS: загрузить файлы в корень сайта (nginx/Apache не требуют настройки).
- Netlify/Vercel/Cloudflare Pages: перетащить папку — получите HTTPS-домен.

Свой домен (например, `smartstock-erp.ru`) подключается у любого из них.
